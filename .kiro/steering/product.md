# Product Overview

Craftus is an AI-powered "Infinite Craft Workbench" that transforms craft ideas into visual, step-by-step instruction sequences. The system generates studio-quality reference images and isolated instruction visuals that recreate the aesthetic of traditional paper craft manuals.

## Core Value Proposition

Users describe a craft project in natural language, select a category, and receive:
- A Master Reference Image showing the finished craft
- A complete materials list
- Step-by-step instructions with isolated "knolled" images showing only the components needed for each step

## Supported Craft Categories

1. Papercraft
2. Clay
3. Fabric/Sewing
4. Costume & Props
5. Woodcraft
6. Jewelry
7. Kids Crafts
8. Tabletop Figures

## Key Features

- **Infinite Canvas**: Spatial workspace for exploring craft projects with pan/zoom
- **AI Generation Pipeline**: Three-stage process (Master Image → Dissection → Step Images)
- **Project Management**: Save, load, and organize craft projects locally
- **Community Gallery**: Share and explore projects in readonly mode

## Design Philosophy

- Resurrect the tradition of old paper instruction sheets with modern AI
- Extreme visual isolation in step images to eliminate confusion
- Dark mode technical blueprint aesthetic
- Minimal backend requirements for hackathon MVP
